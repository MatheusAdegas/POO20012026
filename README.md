Como informado ao professor, eu fiz o sistema do trabalho proposto na minha linguagem de preferência: Luau, sobre a autorização dele. Eu também avisei que seria em uma game engine, o que foi permitido também. Obrigado professor! O principal diferencial da engine é a utilização de [metatables](https://create.roblox.com/docs/luau/metatables) que são uma melhoria das tabelas comuns de .py e similares. </br>
Deixando claro que, mesmo que o sistema fosse bem implementado, a engine só pode rodar com a instalação do Roblox Player ou na IDE do Roblox studio. Por ser uma game engine, eu não usaria isso para o produto de um sistema real. A escolha da engine nesse trabalho foi para expressar minha criatividade e para ser um pouco diferente dos outros trabalhos. </br>
